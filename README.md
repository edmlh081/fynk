## 电子签名的三个级别

Fynk支持eIDAS标准的三种电子签名：

**简单电子签名（SES）**：适合日常商务文件，快速便捷
**高级电子签名（AES）**：更高安全级别，适合重要合同
**合格电子签名（QES）**：最高安全级别，法律效力等同于手写签名

不同场景选不同级别，灵活又合规。而且所有签名都有时间戳和审计追踪，法律效力有保障。
